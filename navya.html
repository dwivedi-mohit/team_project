Java Source Example

This example shows a simple Java program that prints a greeting message. Save it as NavyaHello.java and run it from the command line.

public class NavyaHello {
  public static void main(String[] args) {
    String name = "Navya";
    int year = 2026;
    System.out.println("Hello, " + name + "! Welcome to " + year + ".");
  }
}

Run with: javac NavyaHello.java then java NavyaHello.
